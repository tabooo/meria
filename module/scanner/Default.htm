4. How to extend your license

If your trial license expires, you can contact us to get an extended license. With the extended license, you can generate a new product key and replace the old one in your application:

  • Ask for a license extension: Live Chat or support@dynamsoft.com.
  • Generate a Product Key.
  • Replace ProductKey in JavaScript. The code is typically like this:
    var _dwtParam = {
        'productKey': '******',
        'containerID': 'dwtcontrolContainer'
